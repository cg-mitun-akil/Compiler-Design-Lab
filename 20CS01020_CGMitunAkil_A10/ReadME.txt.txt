1. Execute the yakk and lex files.
2. Write your input in the input.txt.
3. Compile and execute the c file generated.
4. The output will be written in out.txt.

The lex and parser supports
1. variables
2. if else
3. while loop
4. global end
5. print
6. read 
7. + - / * 
8. all relop
9. mod , not , or , and  like c++
